## Lab Description
This lab focuses on building a console-based logistics auditing tool using C#.  

The program collects driver profile information, validates user input, tracks fuel expenses for a 5-day work week, calculates fuel efficiency ratings, and analyzes whether the driver stayed within the weekly fuel budget.

## Program Features
Input validation for fuel budget and total distance
5-day fuel expense tracking
Fuel efficiency calculation
Budget analysis (under budget or exceeded)
Professional audit report output
